What is the Purpose of This Dashboard?

The EY Kickstarter Predictor is a data analytics tool designed to help clients strategically launch their own Kickstarter fundraiser based on historical campaign data.

How Was This Dashboard Built?

This dashboard was built using the R package Shiny by R Studio. Analysis and graphical outputs were created using the open source statistical programming language R.
